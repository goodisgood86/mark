# 📥 앱 다운로드 방법

## 🎯 상황별 다운로드 방법

### 방법 1: TestFlight로 테스트 (베타 테스트)

#### 내부 테스터 (개발자 본인)
1. **iPhone/iPad에서 TestFlight 앱 설치**
   - App Store에서 "TestFlight" 검색 후 설치

2. **테스터 초대**
   - App Store Connect > TestFlight
   - **내부 테스터** 섹션
   - **+** 버튼 클릭
   - 본인 이메일 주소 추가
   - 초대 이메일 발송

3. **초대 이메일 확인**
   - 이메일에서 "View in TestFlight" 클릭
   - 또는 TestFlight 앱에서 초대 코드 입력

4. **앱 설치**
   - TestFlight 앱에서 "Install" 버튼 클릭
   - 앱이 설치됨

#### 외부 테스터 (다른 사람들)
1. **외부 테스터 추가**
   - App Store Connect > TestFlight
   - **외부 테스터** 섹션
   - **+** 버튼 클릭
   - 테스터 이메일 추가
   - 초대 이메일 발송

2. **테스터가 받는 방법**
   - 초대 이메일 확인
   - TestFlight 앱 설치
   - "View in TestFlight" 클릭
   - 앱 설치

---

### 방법 2: App Store에 출시 (공식 출시)

#### 검토 제출
1. **App Store Connect > 앱 스토어**
2. **버전** 페이지에서:
   - 빌드 선택 (Build 2)
   - 앱 정보 입력 완료
   - 스크린샷 업로드
   - 검토 제출

3. **Apple 검토 대기**
   - 보통 1-3일 소요
   - 검토 완료 시 이메일 알림

#### 출시 후 다운로드
1. **App Store에서 검색**
   - iPhone/iPad에서 App Store 열기
   - "Petgram" 검색
   - "받기" 버튼 클릭

2. **또는 직접 링크**
   - App Store Connect에서 앱 URL 확인
   - 링크 공유

---

## 📋 단계별 가이드

### TestFlight로 먼저 테스트하기 (권장)

#### 1단계: 내부 테스터 추가
1. App Store Connect > TestFlight
2. **내부 테스터** 섹션
3. **+** 버튼 클릭
4. 이메일 주소 입력
5. **초대** 클릭

#### 2단계: iPhone에서 설치
1. TestFlight 앱 설치 (App Store에서)
2. 초대 이메일 확인
3. "View in TestFlight" 클릭
4. 또는 TestFlight 앱에서 초대 코드 입력
5. "Install" 버튼 클릭

#### 3단계: 앱 사용
- 설치된 앱 실행
- 모든 기능 테스트
- 문제 발견 시 수정 후 새 빌드 업로드

---

### App Store에 출시하기

#### 1단계: 앱 정보 입력
1. App Store Connect > 앱 스토어
2. **버전** 페이지
3. 다음 정보 입력:
   - 앱 미리보기 (설명)
   - 키워드
   - 스크린샷 (최소 1장)
   - 지원 URL
   - 개인정보 처리방침 URL (필요한 경우)

#### 2단계: 빌드 선택
1. **빌드** 섹션
2. **+** 버튼 또는 "빌드 선택" 클릭
3. 업로드한 빌드 선택 (Build 2)

#### 3단계: 인앱 구매 연결
1. **앱 내 구입 및 구독** 섹션
2. **+** 버튼 클릭
3. `donation_1000` 선택

#### 4단계: 검토 제출
1. 모든 필수 정보 입력 확인
2. **검토를 위해 제출** 클릭
3. 확인 다이얼로그에서 **제출** 클릭

#### 5단계: 검토 대기
- Apple 검토: 1-3일
- 검토 완료 시 이메일 알림
- 승인되면 자동으로 App Store에 출시

#### 6단계: 다운로드
- App Store에서 "Petgram" 검색
- "받기" 버튼 클릭
- 설치 완료

# 📱 Petgram v1.0.0+8 Release Notes

## 🎉 주요 변경사항

### 카메라 시스템 개선
- **실기기 카메라 초기화 문제 해결**
  - `viewId` 설정 순서 수정으로 실기기에서 카메라가 정상적으로 초기화되도록 개선
  - 네이티브 초기화 타임아웃을 10초로 증가하여 권한 요청 대기 시간 고려
  - 초기화 실패 시 상세한 에러 메시지 제공

### 코드 품질 개선
- 불필요한 import 제거
- Deprecated API 수정 (`withOpacity()` → `withValues(alpha:)`)
- 디버그 오버레이 비활성화 (프로덕션 빌드)
- 린터 경고 정리

## 🐛 버그 수정

1. **실기기에서 Mock 카메라가 표시되는 문제**
   - `NativeCameraPreview`의 `onCreated` 콜백에서만 초기화하도록 수정
   - `viewId`가 설정된 후에만 카메라 초기화가 진행되도록 보장

2. **카메라 초기화 실패 시 에러 메시지 개선**
   - 네이티브 초기화 로깅 강화
   - 타임아웃 시 명확한 에러 메시지 제공
